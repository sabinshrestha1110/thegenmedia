# The Gen Media — Website

This repository contains the static website for **The Gen Media: Institute & Studio**.

## Structure
- `index.html` — Home (one-page long)
- `courses.html` — Courses details
- `studio.html` — Studio services
- `careers.html` — Jobs & apply links
- `contact.html` — Contact & forms
- `styles.css` — All CSS
- `scripts.js` — Small JS (nav, dark mode, scroll reveal)
- `favicon.jpg` — Site icon (already uploaded)

## Deployment
This site is static and works on Cloudflare Pages. Commit to `main` and Cloudflare Pages will auto-deploy.
